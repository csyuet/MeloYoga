# MeloYoga

Class booking mobile app for a Yoga Studio
- UI/UX design: Figma
- Implementation: SwiftUI, JSON

### Provides UI of features such as:
- Class schedule tracking
-	Class booking and canceling
-	In-app purchase
-	Record history tracking
